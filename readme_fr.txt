Installez le module Nirror par la méthode courante de téléchargement de fichier ZIP dans la section Modules de votre boutique
Obtenez un compte gratuit auprès du service Nirror en cliquant sur le bouton "Authentification vers l'application Nirror" dans les réglages du module
Affinez les réglages du module puis cliquez sur 'Enregistrer'. C'est installé !
